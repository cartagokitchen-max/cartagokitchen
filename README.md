## 🚀 كيفية الرفع على GitHub Pages

### الخطوة 1: إنشاء حساب GitHub
1. ادخل إلى [github.com](https://github.com)
2. سجل حساباً جديداً (مجاني)

### الخطوة 2: إنشاء مستودع جديد
1. اضغط على الزر الأخضر "New"
2. اسم المستودع: `cartagokitchen`
3. اجعله **Public** (عام)
4. اضغط "Create repository"

### الخطوة 3: رفع الملفات
1. في صفحة المستودع، اضغط "uploading an existing file"
2. اسحب الملفات الثلاث (index.html, style.css, script.js)
3. اضغط "Commit changes"

### الخطوة 4: تفعيل GitHub Pages
1. اذهب إلى "Settings" → "Pages"
2. في "Source" اختر "Deploy from a branch"
3. اختر "main" branch ومجلد "/ (root)"
4. اضغط "Save"
5. انتظر 2-5 دقائق
6. موقعك سيكون على: `https://yourusername.github.io/cartagokitchen`

### الخطوة 5: ربط الدومين cartagokitchen.com
1. في "Settings" → "Pages" → "Custom domain"
2. اكتب: `cartagokitchen.com`
3. اضغط "Save"
4. GitHub سيُنشئ ملف CNAME تلقائياً

### الخطوة 6: إعدادات DNS في مزود الدومين
1. ادخل إلى لوحة تحكم الدومين (حيث اشتريته)
2. أضف هذه السجلات:

| النوع | الاسم | القيمة |
|-------|-------|--------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | yourusername.github.io |

3. انتظر 24-48 ساعة للتحديث

## ✏️ تعديل المعلومات

قبل الرفع، عدّل هذه المعلومات في `index.html`:

### معلومات التواصل
ابحث عن واستبدل:
- `+21600000000` ← رقم واتساب الحقيقي
- `info@cartagokitchen.com` ← بريدك الحقيقي
- `+216 XX XXX XXX` ← رقم هاتفك
- `تونس - صفاقس` ← موقعك الحقيقي

### الصور
الصور الحالية من Unsplash (مجانية). لاستبدالها:
1. ارفع صورك إلى مجلد `images/` في المستودع
2. غيّر الروابط في index.html
